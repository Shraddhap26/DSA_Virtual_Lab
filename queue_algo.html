Algorithm

  • Step 1- Initialize an empty queue
  • step 2 - Enqueue operation: To add an element to the back of the queue, do the following:
    1. Check if the queue is full. If the queue is full, return an error.
    2. If the queue is not full, add the new element to the back of the queue.
  • step 3 - Dequeue operation: To remove an element from the front of the queue, do the following:
    1. Check if the queue is empty. If the queue is empty, return an error.
    2. If the queue is not empty, remove the element from the front of the queue and return it.
  • step 4 - Peek operation: To get the element at the front of the queue without removing it, do the following:
    1. Check if the queue is empty. If the queue is empty, return an error.
    2. If the queue is not empty, return the element at the front of the queue.
  • step 5 - Size operation: To get the number of elements currently in the queue, do the following:
    1. Return the current number of elements in the queue.

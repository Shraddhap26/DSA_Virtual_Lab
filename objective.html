Objective


  • To understand basic techniques and strategies of algorithm analysis for various data structures like array, linked list, stack, queue etc
  • To explain the operation and implementation of data structures. The course teaches students how to implement data structures using various programming languages and explains the operations performed on them.
  • To identify the appropriate data structure and algorithm design method for a specified application.
  • To understand various data searching and sorting methods with pros and cons
